# Method Comparison

## Z-Score

Statistical approach based on standard deviation.

Detected only extreme anomalies.


## Rolling Standard Deviation

Detects local deviations using a 7-day moving window.

Useful for identifying short-term abnormal behavior.


## Isolation Forest

Machine learning algorithm based on decision trees.

Detected more anomalies than statistical methods.



## Autoencoder

Deep learning approach.

Detected the highest number of anomalies and was the most sensitive method.


## Final Conclusion

Z-Score detected only large spikes.

Isolation Forest detected more complex anomalies.

Autoencoder was the most sensitive model.
